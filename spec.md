# Specification

## Summary
**Goal:** Create a mobile-first, responsive FinPocket marketing website with a landing page and supporting informational pages.

**Planned changes:**
- Implement frontend routes/pages: Home, Features, About, Privacy Policy, FAQ.
- Build the Home landing page with the 7 specified sections in order, using the exact provided copy and a “Download (Coming Soon)” CTA plus an app mockup placeholder image.
- Build the Features page with a consistent set of simple icons and short descriptions for the 5 core features.
- Build the About page with the provided mission/positioning copy (verbatim).
- Build the Privacy Policy page with the provided bullet statements (verbatim).
- Build the FAQ page with the provided Q&A items (verbatim) in an easy-to-scan format.
- Add a site-wide footer on all pages with links to all 5 pages, “© 2026 FinPocket”, and a contact email placeholder.
- Apply a clean, minimal, modern startup-style design with a soft blue/green/white theme and mobile-first responsiveness.
- Add SEO basics: semantic HTML structure, per-page document titles, and relevant meta descriptions.
- Add generated static image assets under `frontend/public/assets/generated` and reference them directly in the UI (hero mockup + feature icons).

**User-visible outcome:** Visitors can navigate a fast, responsive FinPocket marketing site with Home/Features/About/Privacy/FAQ pages, consistent styling, a shared footer, SEO-friendly metadata, and visible mockup/icon imagery (with the download CTA clearly marked as “Coming Soon”).
